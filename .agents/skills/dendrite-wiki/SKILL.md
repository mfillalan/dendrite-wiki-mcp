# Dendrite Wiki

Use this workflow when a project has Dendrite Wiki MCP installed.

1. Read docs/index.md.
2. Ask the MCP server for a wiki_context briefing for the current task.
3. Use wiki_search or wiki_read for relevant pages.
4. Update wiki pages and docs/wiki/project-log.md when durable knowledge changes.
5. Run dendrite-wiki benchmark:snapshot after meaningful sessions when measuring whether the wiki improves agent orientation over time.
